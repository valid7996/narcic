# Narcic Help

## First Login
URL: `https://your-worker.workers.dev/sync/dash`
Default password: `admin`

## D1 Binding Name
Variable name must be exactly: `NARCIC_DB`

## Supported Clients
- v2rayNG
- Clash / Clash Meta
- Sing-Box
- Shadowrocket
- NekoBox

## Subscription URL
After login, go to the **Endpoints** tab to find your subscription URL.
